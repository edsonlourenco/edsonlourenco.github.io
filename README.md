## 🚀 Featured Projects

Here are the main projects you will find detailed in my portfolio:

### 1. 🌌⚛️ [Quantum Computing Studies - FADERGS](https://github.com/edsonlourenco/pos-computacao-quantica-fadergs)
My study repository containing notes, mind maps, and practical exercises developed during my postgraduate program in Quantum Computing. Perfect for anyone who wants to see the practical application of quantum physics and programming quantum computers.

### 2. 🦾🤖 [Vector-Brain](https://github.com/edsonlourenco/Vector-Brain)
An intelligent virtual assistant (chatbot) that reads digital books (PDFs) and answers questions based on their content. It utilizes Artificial Intelligence (LLMs) and text vectorization.

### 3. 🚀🌌 [Dark-Matter](https://github.com/edsonlourenco/Dark-Matter)
A complete Data Engineering home lab running on your computer using Docker containers. It is an excellent ecosystem for studying and testing the processing of large volumes of data (Big Data), managing data lakes, and task orchestration.

### 4. 📚 [Fake Books Dataset Generator](https://github.com/edsonlourenco/pyndantic-usecase-example)
An automatic synthetic dataset generator for fake books developed in Python. It is very useful for creating realistic test data using the `Faker` and `Pydantic` libraries.
