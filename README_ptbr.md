## 🚀 Projetos em Destaque

Aqui estão os principais projetos que você encontrará detalhados no meu portfólio:

### 1. 🌌⚛️ [Estudos de Computação Quântica - FADERGS](https://github.com/edsonlourenco/pos-computacao-quantica-fadergs)
Meu repositório de anotações, mapas mentais e exercícios práticos desenvolvidos durante a pós-graduação em Computação Quântica. Ideal para quem quer ver a aplicação prática de física quântica e programação de computadores quânticos.

### 2. 🦾🤖 [Vector-Brain](https://github.com/edsonlourenco/Vector-Brain)
Um assistente virtual inteligente (chatbot) que lê livros digitais (PDFs) e responde a perguntas baseando-se no conteúdo deles. Ele utiliza Inteligência Artificial (LLMs) e vetorização de textos.

### 3. 🚀🌌 [Dark-Matter](https://github.com/edsonlourenco/Dark-Matter)
Um laboratório completo de Engenharia de Dados que roda no seu computador usando containers Docker. É um ecossistema excelente para estudar e testar o processamento de grandes volumes de dados (Big Data), organização de bancos de dados (Data Lakes) e orquestração de tarefas.

### 4. 📚 [Fake Books Dataset Generator](https://github.com/edsonlourenco/pyndantic-usecase-example)
Um gerador automático de dados de livros fictícios desenvolvido em Python. É super útil para criar dados de teste realistas usando as bibliotecas `Faker` e `Pydantic`.
